# Glossário de domínio

Manter **um termo canónico** por conceito; variantes entram em «Sinónimos evitados» ou em `00-Ambiguidades.md` até haver decisão.

| Termo canónico   | Definição | Sinónimos evitados                                | Notas |
| ---------------- | --------- | ------------------------------------------------- | ----- |
| _ex.: Encomenda_ | _…_       | Order (se «Order» for reservado a outro conceito) |       |
|                  |           |                                                   |       |

## Regras

- Antes de introduzir **novo** termo de domínio em Stories/Tasks, **consultar** este ficheiro.
- Conflitos (ex.: Order vs Purchase): seguir o glossário ou **registar** ambiguidade até o PO fechar.
