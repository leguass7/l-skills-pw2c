# Example Skill

Use esta skill para validar rapidamente o fluxo do `l-skills-pw2c`.

## Objetivo

- confirmar que a skill foi instalada na pasta do projeto
- servir como base para criação de novas skills do time
- demonstrar como incluir referências e templates

## Como usar

Peça ao agente para seguir a checklist abaixo:

1. Ler o problema com atenção.
2. Resumir o objetivo em uma frase.
3. Propor uma solução pequena antes de editar.
4. Validar a saída com teste ou comando adequado.
